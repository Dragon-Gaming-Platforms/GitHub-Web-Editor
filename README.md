## Overview
This is a Editor for GitHub Repositories and [Codespaces](https://github.com/features/codespaces) that uses a PAT to interact with the user's account. The editor comes complete with Monaco editor, GitHub Actions workflows, and much more! [Check it out here](https://dragon-gaming-platforms.github.com/Github-Web-Editor/)

## Self-Hosting
If you would like to host your own deployment of this project simply fork the repository, navagate to Pages settings and enable pages on ```main``` and ```(root)/```
